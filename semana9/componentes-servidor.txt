Sistema operativo:
Es el entorno donde se ejecuta el servidor y gestiona los recursos del sistema.

Node.js:
Es el entorno que permite ejecutar JavaScript en el servidor.

Aplicación server.js:
Es el archivo donde se define la lógica del servidor y las rutas.

Puerto:
Es el punto de entrada por donde el servidor recibe solicitudes (en este caso el 3000).

Logs:
Son registros que muestran lo que ocurre en el servidor para poder monitorear errores o eventos.
