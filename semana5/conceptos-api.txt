La ruta POST /reporte recibe información enviada por un usuario mediante una solicitud HTTP. En este caso recibe un mensaje y devuelve una respuesta en formato JSON confirmando que el reporte fue recibido.

En una aplicación comunitaria esta ruta podría utilizarse para registrar reportes ciudadanos, alertas de seguridad, solicitudes de ayuda o información relacionada con actividades de la comunidad. De esta manera la información puede organizarse y procesarse de forma estructurada.
