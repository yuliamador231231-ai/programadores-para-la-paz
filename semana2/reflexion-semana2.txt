Aprender a utilizar la terminal es importante para cualquier desarrollador porque permite interactuar directamente con el sistema operativo de forma rápida y eficiente. A través de la terminal es posible navegar entre carpetas, crear archivos y organizar proyectos sin depender únicamente de herramientas gráficas. Además, muchas tecnologías y entornos de desarrollo utilizan comandos para automatizar tareas. Conocer estos comandos ayuda a trabajar de manera más profesional y a comprender mejor la estructura de los proyectos. También facilita el uso de herramientas como Git y GitHub para el control de versiones. Durante esta actividad comprendí cómo crear carpetas y archivos desde la terminal y cómo mantener organizada la información. Considero que esta habilidad será muy útil en mi proceso de formación como desarrolladora. Cada práctica realizada fortalece mi confianza para trabajar en proyectos reales. La terminal es una herramienta fundamental para mejorar la productividad y el control del trabajo realizado.
